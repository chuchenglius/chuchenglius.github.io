This image is optimized from the Squoosh website. I chose the Squoosh because it has many option items, so I could try many solutions. If I don't change the image size and quality is 65%, the file size is 233 KB. I thought that is too big, so I reduced the image size to 50%. Finally, the file size to 76.1 KB. When we compare two files, we could save the file size by 88% and save the transfer time by 81.63%.
