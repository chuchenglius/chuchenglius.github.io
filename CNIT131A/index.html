CNIT-131A - XML and JSON Assignments

Item Description of the assignment
Homework2 "Your first XML" is the first XML homework in this course.
Homework3 "XSLT" is the second homework in this course. It has some targets follow as:
1. You will present a h1 that says: List of Clients
2. You will present the clients in a table presenting the name, phone, email, account_total
3. The account_total value should be right-aligned within the column
4. When the account_total is less or equal to 80000, the number should be presented in red color and bold.
5. There should be a $ (dollar sign) in front of the account_total value when presented on the table
Homework4 "XPath, XSL-FO" is this week's topic. It has some targets follow as:
1. You will insert in this new XML file (hw4.xml), for each Client, a new element that you will call <Years>x</Years> where x will be a number between 1 and 10 that would show how many years that person is a client for ABC Financial Startup company. Please, make this x be greater than 7 at least for one or a couple of clients!
2. You will create the following as required in the new XSL file (hw4.xsl).
2-1. The first line is a simple h1 element
2-2. Add the name of the image is financialstartup.jpg and the image is centralized within the width of the browser.
2-3. The paragraph below the image is a simple paragraph.
2-4. The second paragraph uses what was learned in XPath to count the number of clients the company currently has
2-5. The third paragraph uses also what has been learned to list the names of the clients separated by , (comma), but the last name will come preceded by , and (comma and word and).
2-6. The last paragraph also uses what has been learned in XPath to count the clients that have been with the company for more than 7 years!
Homework5 "DTD" is this week's topic. It has some targets follow as:
1. You will use the hw5.xml to complete this assignment.
2. You will create a DTD file (external file, called hw5.dtd) or an internal DTD (inside the hw5.xml) based on the following assumptions that I'm sure you will be able to detect by reading the hw5.xml code.
2-1. The ProdID attribute is required and is unique (cannot be repeated to any other product inserted in the XML file)
2-2. The Color element is optional
2-3. Notice that "Made in USA" shows up in every product, which means it can be declared as an Entity in the DTD
2-4. For the XML file to be considered valid, you need to have at least one product listed
3. Based on the rules you will specify in the DTD, I will be able to create a new product in the XML file and the new product will be accepted as valid or invalid based on the rules of the DTD
Homework6 "Schema" is this week's topic. It has some targets follow as:
1. For this homework, you will use the hw6.xml(you can find this file inside the CNIT131AFiles.zip folder you have downloaded).
2. You will create the Schema (hw6.xsd) file based on the following assumptions:
2-1. The Name should only accept a maximum of 30 characters
2-2. The Age should have a minimum value of 21 and if nothing is stated for age, the default value should be 21
2-3. The Date should be in that format that you see in hw6.xml file
2-4. The Phone should always be using that pattern of 3 digits - 3 digits - 4 digits
3. You need to make sure that you will be able to validate the XML file against the XSD - see the reference to some free online validators given below
Homework7 "Namespace" is this week's topic. It has some targets follow as:
1. You will put hw7.xml and hw7a.xml together in a file named hw7final.xml and, one of the namespaces should be declared as the default and the other one with a prefix of your choice.
2. Then, put all the elements from both files in the same file and use the prefix accordingly to differentiate states from the east and west part of the country.
3. You will create a CSS file that you will name hw7.css, to present the final result of this assignment and the CSS should format the data with the following styles:
3-1. Each state should be presented in a different line
3-2. The states from west should be presented with a different background color (not the same background color of the state from the east)
TIP: As you will need to use namespace in the CSS file as well, you should use the @namespace prefix URL in the first line of your CSS file to declare the namespace you will be using in your CSS selector. Then, whenever you want to use the prefix for a certain element of the XML, you should use it following this format prefix|element
