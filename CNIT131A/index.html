Homework2 "Your first XML" is the first XML homework in this course.
Homework3 "XSLT" is the second homework in this course. It has some targets follow as:
1. You will present a h1 that says: List of Clients
2. You will present the clients in a table presenting the name, phone, email, account_total
3. The account_total value should be right-aligned within the column
4. When the account_total is less or equal to 80000, the number should be presented in red color and bold.
5. There should be a $ (dollar sign) in front of the account_total value when presented on the table
Homework4 "XPath, XSL-FO" is the second homework in this course. It has some targets follow as:
1. You will insert in this new XML file (hw4.xml), for each Client, a new element that you will call <Years>x</Years> where x will be a number between 1 and 10 that would show how many years that person is a client for ABC Financial Startup company. Please, make this x be greater than 7 at least for one or a couple of clients!
2. You will create the following as required in the new XSL file (hw4.xsl).
2-1. The first line is a simple h1 element
2-2. Add the name of the image is financialstartup.jpg and the image is centralized within the width of the browser.
2-3. The paragraph below the image is a simple paragraph.
2-4. The second paragraph uses what was learned in XPath to count the number of clients the company currently has
2-5. The third paragraph uses also what has been learned to list the names of the clients separated by , (comma), but the last name will come preceded by , and (comma and word and).
2-6. The last paragraph also uses what has been learned in XPath to count the clients that have been with the company for more than 7 years!
